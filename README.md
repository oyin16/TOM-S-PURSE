# TOM'S PURSE WEBSITE
This is an implementation of Tomsparse website, a fintech platform designed to simplify investment access for users. This project is built using ReactJS,based on the provided Figma designs.

Pages Implemented;
- Landing Page – A responsive homepage introducing Tomsparse and its value proposition.
- About Us Page – Highlights the mission and vision of Tomsparse.
- Sign Up Page – Allows new users to create an account.
- Login Page – Enables existing users to securely access their dashboard.

Tech Stack;
- ReactJS – Core framework for building the UI.
- React Router – For seamless page navigation.
- CSS/Styled Components – For responsive design and styling.

All implemented pages are fully responsive across mobile, tablet, and desktop views, ensuring a smooth user experience on any device.
